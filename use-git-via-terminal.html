How to Upload a Project to GitHub

Initialize a Local Git Repository:

Navigate to project directory, run git init

Add Project Files to the Repository:

Run git add . or git add <file1> <file2> ...

Create a .gitignore File (Optional):

Create .gitignore, list files to exclude

Commit the Changes:

Run git commit -m "Initial commit"

Create a New Repository on GitHub:

Log in to GitHub, create new repo, keep it empty

Link the Local Repository to the GitHub Repository:

Run git remote add origin <repository_URL>

Push Local Repository to GitHub:

Run git push -u origin master

Subsequent Pushes:

For updates, commit and run git push origin <branch_name>
